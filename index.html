Menu Master

Objetivo

El proyecto tiene como objetivo mejorar la eficiencia y la experiencia del cliente en los restaurantes, mediante la introducción de un sistema de toma de pedidos digital. El sistema permitirá a los camareros crear y acceder a sus cuentas, desde ahí podrán controlar y tomar los almuerzos en las mesas, brindando un servicio eficiente y preciso.

Pantallas

Se desarrollará una aplicación utilizando Dart y Flutter para la gestión de restaurantes. Los camareros pueden registrarse y gestionar sus cuentas, actualizar su perfil y configurar su disponibilidad. Los restaurantes pueden actualizar el menú del día para mostrar a los clientes las opciones disponibles. Los camareros pueden tomar y servir pedidos, así como ofrecer ofertas especiales de manera eficiente. Además, se integrará una impresora de cocina para acelerar la preparación de alimentos.

La aplicación contará con una interfaz de usuario atractiva y se implementará la lógica de la aplicación en Dart. Se configurará una base de datos para almacenar información relevante, y se establecerán conexiones para la comunicación en tiempo real entre la aplicación de camareros y la impresora de cocina.
